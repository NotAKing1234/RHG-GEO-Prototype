## S1273 - schema.org
- URL: https://schema.org/Hotel
- Finding group: GAP-003
- Assessment: Every hotel/property URL should expose visible facts plus JSON-LD using Hotel/LodgingBusiness: name, URL, @id, address, geo, phone, images, check-in/out, room count where available, amenities, pets/parking/Wi-Fi policies, ratings only when visible and compliant, and direct booking/offer fields where accurate.
